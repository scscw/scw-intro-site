## 🚀 실행 방법 (Windows 기준)

1. 저장소 클론
git clone https://github.com/yourusername/scw-intro-site.git
cd scw-intro-site

2. 가상환경 생성 및 실행
python -m venv myproject
myproject\Scripts\activate

3. 패키지 설치
pip install -r requirements.txt

4. 서버 실행
cd mysite
python manage.py runserver

5. 웹 브라우저에서 접속
http://127.0.0.1:8000/
